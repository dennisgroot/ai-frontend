# Raahuis boilerplate

Dit boilerplate maakt gebruik van de volgende technieken:

- 11ty (eleventy) + Vite voor front-end routing en HTML-partials
- Tailwind CSS - Utility-first CSS framework
- Alpine JS - Minimaal Javascript framework voor overzichtelijke en sneller javascript.
- Raadhuis Gutenberg blocks - Eigen gebouwde blokken door Raadhuis
- SASS compiler - Voor eventueel sneller schrijven van bijvoorbeeld Tailwind componenten

### Andere features

- Navigatie plugin met gegevensbestand voor het configureren van HTML
- SEO geoptimaliseerd meta-inclusiebestand met meerdere fallbacks
- Afbeeldingsplugin voor prestaties
- Generatie van sitemap

## Benodigdheden

- [Node.js](https://nodejs.org/en/download/package-manager) versie 20 of hoger
- [Yarn (package-manager)](https://yarnpkg.com/getting-started/install) versie 1.22 of hoger
- Optioneel: [Laravel valet](https://laravel.com/docs/11.x/valet#installation) (in het geval van back-end
werkzaamheden)
- Optioneel: [Homebrew](https://brew.sh/nl/) - Handige package manager
- Optioneel: [NVM (Node Version Manager)](https://medium.com/@priscillashamin/how-to-install-and-configure-nvm-on-mac-os-43e3366c75a6) - Om te switchen van Node versie binnen een project.

## Instalatie

Voer onderstaande commands uit:

1. yarn set version stable
2. yarn install
3. yarn start
